# RupPub
一个持续发布更新工具(Console)


/tz "2014-07-01,2014-07-15" 时间区间，英文逗号分隔时间值。
/am "2014-07-01 08:00:00" 在某个时间之后修改的。
/ske ".pdb;.cs;.txt" 忽略的文件扩展名列表
/ice ".config;.xml"  需要包含文件扩展名列表
-h   忽略带隐藏属性的文件
/skp "obj;Reference;Libs" 忽略的相对目录列表
/zfile 生成的zip发布包文件名称
/fdir "获取文件目录路径"
/pdir "发布文件目录路径"
/show 查看满足条件的所有文件
